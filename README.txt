Ломает дерево полностью если топор в руках. Дерево - бревно + блок листвы.

Если в руках держать ножницы - листва также автоматически ломается. Максимальное расстояние блока листвы от бревна настраивается в конфиге.
Также в конфиге указываются все виды бревен, листов, топоров и ножниц.
Инструменты сломаются не уничтожив блок если прочность станет равна 0.

Механика казино - если ломаемое топором дерево выше определенного количества блоков - каждый сломанный блок превращается в лудку.
Шанс лудки, необходимая высота дерева, награды и их количества настраиваются в конфиге.

Конфиги:
-main:
-- applicableLogs - Бревна, которые считаются в дерево
-- applicableLeaves - Листья, которые считаются в дерево
-- applicableAxes - Топоры, которые могут срубить дерево
-- applicableShears - Ножницы, которые могут срезать листву
-- maxLeafDist - Максимальное расстояние от блока бревна до блока листвы. Если листва дальше указанного блока - она не будет ломаться при поломке дерева
-- treeHeightForCasinoActivation - Минимальная высота дерева для активации механики казино
-- casinoLooseChance - Шанс проиграть в казино
-- casinoItems - Шафлбэги для каждого из вида деревьев. Формат "<блок дерева>": "<Название шафлбэга>"

-shuffle-bags - Хранит в себе шафлбэги. Формат:
"<Название шафлбэга>": [
    {
      "loot": "<Название шафлбэга>", // Тип "лут" означает что данный вариант ссылается на шафлбэг из конфига fillers
      "bag": "<Название шафлбэга>", // Тип "bag" означает что данный вариант ссылается на шафлбэг из конфига shuffle-bags
      "amount": 50, // Количество данных вариантов
      "stack": {
        "id": "<Айди предмета>",
        "stackSize": <Количество предмета>
      }
    }
]

-fillers - Хранит в себе шафлбэги. Отличается от shuffle-bags тем, что блоки из main.casinoItems не могут напрямую ссылаться на шафлбэги из fillers
